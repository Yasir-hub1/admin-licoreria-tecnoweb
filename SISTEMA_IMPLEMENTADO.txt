================================================================================
COMANDOS PARA INICIAR
================================================================================

# 1. Instalar dependencias (si no lo hiciste)
composer install
npm install

# 2. Configurar .env
cp .env.example .env
# Editar .env con tu configuración de base de datos

# 3. Generar key
php artisan key:generate

# 4. Ejecutar migraciones
php artisan migrate

# 5. Crear seeders y ejecutar
php artisan db:seed

# 6. Compilar assets
npm run dev

# 7. Iniciar servidor
php artisan serve

# Visitar: http://localhost:8000

================================================================================
NOTAS IMPORTANTES
================================================================================

1. El sistema usa SESIONES para el carrito, no requiere autenticación
2. Al hacer login, el carrito de sesión se puede migrar al usuario
3. Los roles son: propietario, vendedor, cliente
4. Las rutas admin requieren roles propietario o vendedor
5. Las rutas de checkout requieren rol cliente
6. El modelo Usuario extiende Authenticatable
7. Las contraseñas se hashean automáticamente
8. Inertia comparte auth.user y flash messages globalmente
